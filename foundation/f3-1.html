Prompting fundamentals

F3.1  ·  part of the Foundation track

Objective

Learn a repeatable prompting method so you get reliable results instead of lucky ones.

Key concepts

  • A simple framework: Task, Context, References, Evaluate, Iterate
  • Role, format, and constraints in a prompt
  • Few-shot prompting: showing examples
  • Giving the model a way to say “I don't know”

Hands-on

Take a weak one-line prompt and improve it through five iterations, saving each version to start a personal prompt library.
